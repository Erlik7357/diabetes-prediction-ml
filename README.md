# diabetes-prediction-ml
Classification of diabetic and non-diabetic by using Support Vector Machine algorithm

# Install Required Libraries

```bash
pip install -r requirements.txt
```

Or manually install:

```bash
pip install numpy pandas scikit-learn
```

# Run the Model

Open the Python file or notebook and run:

```bash
python diabetes_prediction.py
```

Enter input values when prompted, and the model will display the prediction.
